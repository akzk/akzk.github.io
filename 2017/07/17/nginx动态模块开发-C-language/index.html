nginx动态模块开发(C language)

发表于 2017-07-17 | 分类于 代码

使用

  1. 套用模版

    复制模版文件夹，将所有”xxx”替换为开发者自己命名的模块名称，包括文件名与文件内容

  2. 编写业务

    为ngx_http_xxx_module.c模版对应的新命名C文件编写业务逻辑，见”// todo”处

  3. 安装nginx并编译、加入动态模块

    nginx可使用yum安装，版本要求大于1.9.11，另下载对应版本源码，用于编译动态模块
