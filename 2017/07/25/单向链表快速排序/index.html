单向链表快速排序

发表于 2017-07-25 | 分类于 代码

简述

单向链表快速排序与数组快速排序$^1$有所不同，关键部分中，数组快速排序有不断向前的下标访问，这种方式对于双向链表来说可行，但对于单向链表来说，每次访问都会导致大量解引用行为，因此需要对方案进行适度的修改。

单向链表快速排序有两种方案，改变值、改变指针，改变值的方案在节点结构体包含大量变量内容的情况下会造成性能大幅下降，故选择后者。
